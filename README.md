# Welcome to the Home Repo for Web3 Law

This repo contains materials and guidance for a law school course on Web3 law.  Currently, I am teaching this class at the University of Houston Law Center.  The repo is divided into 28 folders to represent the 28 class periods for a semester.  Each folder contains a jupyter notebook file containing the class outline and links to relevant materials.  Additionally, where available, source materials are included.  Finally, this repo contains a source index that contains all of the sources within the source database file.
