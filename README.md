# Welcome to the Home Repo for Web3 Law

This repo contains materials and guidance for a law school course on Web3 law.  Currently, I am teaching this class at the University of Houston Law Center.  The repo is divided into 28 jupyter notebook files containing the class outline and links to relevant materials.  Additionally, where available, source materials are included for each class.  Finally, this repo contains a source index that contains all of the sources within the source database file.  

## Class 1:  Introduction to the Class Syllabus, GitHub, and Terminal

To prepare for class, you should have read the following:
- The syllabus

[Class 1 Notebook](/1-introduction.ipynb)
